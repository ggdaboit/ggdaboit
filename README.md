# Hey, I'm Gilleady Daboit 👋

- Design and implement multi-account AWS architectures
- Build infrastructure as code with **OpenTofu / Terraform**
- Deploy and manage containers on **EKS** and **ECS**
- Create CI/CD pipelines with **GitHub Actions**
- Package and manage Kubernetes apps with **Helm Charts** and **ArgoCD**
